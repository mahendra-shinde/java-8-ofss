HTML Forms

HTML form allows input from end-user to be submitted to a server side script. Example: Collect a 'text' with name 'q' and sending to URL `https://google.co.in/search` will result in invoking "search" script on Google Search engine and passing the text with name 'q' as an parameter. HTML Form Inputs Default: text Single line text input field password Textfield with masked character either * or DOT radio Radio button (Always in GROUP) checkbox Check Box (Always in GROUP) Within a group only ONE radio can be selected but any number of checkboxes can selected button A Generic Button, which requires JS to perform action submit A Button with pre-defined action "Form Submission" reset A Button with pre-defined action "Form Reset" hidden A Field no visible to end-users. SELECT with OPTION: Drop down list with choices. TEXTAREA: Multi line text field
